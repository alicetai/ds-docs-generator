# Design System Docs Generator

A Figma plugin that generates comprehensive design system documentation for Figma components.

## Installation

1. Download the plugin files
2. Open Figma Desktop → Plugins → Development → Import plugin from manifest
3. Select `manifest.json`

## Usage

1. Select a **component** (not a frame or other object)
2. Run the plugin: Plugins → Development → Design System Docs Generator
3. Click "Generate documentation"
4. Documentation appears as a formatted frame next to your component

## Files

- `manifest.json` - Plugin configuration
- `code.js` - Main plugin logic (Figma API)
- `ui.html` - User interface
